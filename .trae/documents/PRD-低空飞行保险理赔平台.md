## 1. 产品概述

低空飞行保险理赔 Web 平台是面向无人机运营商、保险经纪和理赔专员的全流程保险服务平台，覆盖从投保申请到理赔结案的完整业务链路。通过数字化手段提升投保效率、简化理赔流程、降低运营成本。

- 核心价值：连接无人机运营商、保险机构与理赔服务方，实现低空飞行保险业务的全流程线上化
- 目标市场：民用无人机行业、低空经济领域的保险服务市场

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 无人机运营商 | 企业注册 | 机型载荷登记、投保申请、保单管理、事故报案、材料上传、赔付进度查询 |
| 保险经纪 | 经纪人账号 | 方案比选、在线报价、批量投保、客户管理、续期提醒 |
| 理赔专员 | 内部账号 | 查勘协同、材料审核、赔付测算、结案确认、赔案分析 |

### 2.2 功能模块

1. **投保申请**：机型载荷登记、保障方案比选、在线报价、投保单填写
2. **保单管理**：保单列表、保单详情、批量投保、续期提醒、保单变更
3. **飞行任务关联**：任务创建、风险备注、保单绑定、飞行日志
4. **事故报案**：事故定位、基本信息填报、损失清单、第三方责任登记
5. **材料收集**：照片视频上传、证件材料提交、材料审核状态
6. **查勘协同**：查勘预约、查勘记录、现场照片、查勘报告
7. **赔付进度**：赔付测算、审核节点、付款状态、结案确认
8. **统计台账**：保单统计、理赔统计、赔付分析、争议记录

## 3. 核心流程

### 3.1 投保流程
无人机运营商登记机型信息 → 选择保障方案 → 获取在线报价 → 填写投保单 → 支付保费 → 生成电子保单

### 3.2 理赔流程
事故发生 → 在线报案（定位+损失清单）→ 上传事故材料 → 查勘员现场查勘 → 材料审核 → 赔付测算 → 多级审核 → 赔付支付 → 结案确认

```mermaid
    E --> F["飞行任务"]

    G --> H["在线报案"]
    H --> I["材料上传"]
    I --> J["查勘协同"]
    J --> K["赔付测算"]
    K --> L["审核审批"]
    L --> M["赔付支付"]
    M --> N["结案归档"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝 #0F3460（专业、信赖）+ 橙色 #E94560（警示、行动）
- **辅助色**：浅蓝 #1A5276、灰色 #5D6D7E、成功绿 #27AE60
- **按钮风格**：圆角 6px，悬停有微妙阴影效果，主按钮渐变
- **字体**：标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**：顶部导航 + 左侧菜单 + 内容区域的企业级后台布局，卡片式组件
- **图标风格**：线性图标，统一 24px 尺寸，使用 Phosphor Icons 风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 投保申请 | 表单区域 | 分步向导、卡片式分组、地图选择器、文件上传区 |
| 保单管理 | 数据表格 | 筛选栏、数据表格、操作列、状态标签、分页器 |
| 飞行任务关联 | 任务卡片 | 时间线、风险标签、地图缩略图、状态指示 |
| 事故报案 | 报案表单 | 地图组件、损失清单动态添加、分步进度条 |
| 材料收集 | 上传区域 | 拖拽上传、预览缩略图、上传进度、审核状态标签 |
| 查勘协同 | 日程视图 | 日历组件、查勘员列表、状态流转按钮 |
| 赔付进度 | 流程追踪 | 垂直时间线、金额高亮、审批节点状态 |
| 统计台账 | 数据看板 | 统计卡片、柱状图、折线图、饼图、数据表格 |

### 4.3 响应式
- 采用桌面优先设计，适配 1920px、1440px、1024px 主流分辨率
- 平板端折叠左侧菜单为抽屉式
- 移动端简化表格展示，采用卡片列表形式

### 4.4 动效设计
- 页面切换：淡入 + 轻微上移动画
- 数据加载：骨架屏占位
- 表单交互：输入框聚焦时有边框色过渡
- 按钮点击：scale 0.98 的按压效果
- 状态变化：状态标签颜色渐变过渡
